Place your HTML games in this folder.

Each game should be a single .html file. The game will be auto-discovered and displayed on the hub.

Supported:
- index.html, game.html, or any *.html file
- favicon.ico in the same folder for the game icon (optional)
- Game title and description are read from the HTML file's <title> and <meta name="description"> tags

Example game structure:
  games/
    snake.html
    tetris.html
    pong.html
    2048/
      index.html
